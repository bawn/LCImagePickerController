多选图片库

![demo](demo.gif)

##功能
1. 支持单选、多选
2. 可设置是否默认进入相机胶卷
3. 显示选择顺序标签
4. 支持限制选择数量
5. 自定义样式
6. 隐藏空的相册专辑页
7. 倒序排列
